javascript 編寫和 MIT 條款，功能雖簡單但寫的過程實在艱辛，
在此下略一萬字，喜愛的童鞋們可以到 ./js/acgn.js 裏研究一下
這 140kb 的檔案，想直接用引擎的童鞋們可以直接到 ./script/
001.gs 編輯，語法跟 krkr 大致一樣，稍微研究一下還是可以完
美制作出一個視覺小說的。
